# Number Guessing Game (Java)

## Project Description
This is a simple console-based Number Guessing Game developed in Java.

The program generates a random number between 1 and 100, and the user must guess the number. The program provides hints whether the guess is higher or lower than the secret number.

## Features
- Random number generation
- User input handling using Scanner
- Loop until correct guess
- Attempt counter
- Simple and clean console interaction

## Concepts Used
- Java Basics
- Loops (while)
- Conditional Statements (if-else)
- Random class
- Scanner class

## How to Run

1. Compile:
   javac NumberGuessingGame.java

2. Run:
   java NumberGuessingGame
